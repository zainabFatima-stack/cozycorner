# CozyCorner – A Safe Space for Expression & Community

CozyCorner is a full-stack web application designed to provide users with a safe, welcoming, and supportive online space to express themselves. The platform allows users to share blogs, write anonymous letters, provide feedback, and manage personal profiles while promoting a positive and engaging community.

This project was developed as a learning project to strengthen full-stack web development skills, including frontend design, backend development, database management, and user authentication.

## Features

### User Authentication

* User Registration
* Secure Login & Logout
* Session Management
* Profile Management

### Blogging

* Create blog posts
* Edit existing blogs
* Delete blogs
* View blogs shared by the community

### Anonymous Letters

* Write anonymous letters without revealing identity
* Read anonymous community posts
* Encourage open and honest expression

### Feedback System

* Submit feedback about the platform
* Improve user experience through community suggestions

### User Profile

* Update personal information
* Manage account details
* View personal content
