## 🌸 About CozyCorner

**CozyCorner** is a full-stack web application created to provide a peaceful digital space where people can express themselves without fear of judgment.

The platform encourages creativity, self-expression, and meaningful conversations through blogging and anonymous letters while maintaining a warm, welcoming environment inspired by soft pastel colors and sunflowers.

Whether someone wants to share their thoughts openly, write anonymously, or simply read stories from others, CozyCorner offers a safe and comforting experience.

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/cozycorner.git
```

2. Move the project into your WAMP/XAMPP `www` or `htdocs` folder.

3. Import the SQL database into phpMyAdmin.

4. Start Apache and MySQL.

5. Open your browser and visit:

```
http://localhost/cozycorner
```
